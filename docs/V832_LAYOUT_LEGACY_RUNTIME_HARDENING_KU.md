# AIR-DROW v8.3.2 — ڕێکخستنی layout و بەهێزکردنی براوسەری کۆن

## ئەوەی چاککرا
- کارتی **تەچ • قەڵەم • دەست** لە بەشی دەربارەی ئەپ فراوانتر کرا. لە سۆرانی لە لای چەپ دەردەکەوێت و کارتی وەشان کورت و ڕێک لە لای ڕاست دەمێنێت.
- کاتێک ڕێکخستن بۆ یەکەم جار دەکرێتەوە، scroll دەگەڕێتەوە بۆ سەرەوە؛ کارتێکی نیوە لە ژێر سەرپەڕەوە نابینرێت.
- لە بەشە سەرەکییەکانی کێشان و hand tracking ـدا پشت بە `structuredClone()` و `Array.prototype.at()` نابەسترێت. بۆ WebKit ـی کۆن fallback ـی ناوخۆیی زیادکرا.
- شێوازی تەواوی هێڵی دەستی v7.6.2، پاراستنی export، و تەنها-کرتەی ڕاستەقینە بۆ Save/Export/Clear هەر بەجێ ماون.

## تاقیکردنەوە
- `npm run verify:all` سەرچاوە، زمان، کاش، MediaPipe، export، WebKit و test ـەکان پشکنین دەکات.
- سکریپتی Termux پڕۆژەکە بە شێوەی پارێزراو جێگۆڕ دەکات و فۆنتی کوردیی ناوخۆیی repo ـەکەت دەپارێزێت.
