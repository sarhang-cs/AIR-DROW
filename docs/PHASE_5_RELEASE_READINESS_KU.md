# AIR-DROW v6.4.0 — قۆناغی ٥: ئامادەیی بڵاوکردنەوە و زانیاریی وەشان

ئەم قۆناغە کۆتایی قۆناغەکانی ئامادەکردنی AIR-DROW ـە. گۆڕانکارییەکان لەسەر ناسنامەی وەشان، بەڵگەنامە، پشکنینی بڵاوکردنەوە و پیشاندانی زانیاری لە ناو ئەپ دان؛ ئینجینی کێشان و شوێنکەوتنی دەست نەگۆڕاوە.

- لە Settings > دەربارەی ئەپ، وەشانی ڕاستەقینە و ناسنامەی build بەخۆکار لە runtime ـەوە پیشان دەدرێت.
- وەشان و buildId لە package، PWA، service worker، API health، release.json، asset manifest و project manifest یەکسانن.
- `npm run phase5:verify` ئەگەر یەکێک لە بەڵگەنامەکان، ناونیشانەکان یان UI ـی زانیاری وەشان لە metadata جیابوون، build وەستێنێت.
- هەموو ڕەفتارەکانی Phase ١ تا ٤ دەماونەوە: runtime ـی MediaPipe و مۆدێلی دەست local ـن، کێشانەکان local ـن، export پێش خۆی پاشەکەوت دەکات و update پێش نوێبوونەوە save دەسەلمێنێت.

## پشکنینی دوای deploy

١. Settings بکەرەوە و بچۆ دەربارەی ئەپ.
٢. دڵنیابە `v6.4.0` و `air-drow-v640-phase5-release-readiness` دیارن.
٣. بە touch هێڵێک بکێشە، export ـی PNG یان SVG بکە.
٤. ئەگەر کامێرا پێویستە، تەنها لە کاتی کردنەوەی کامێرا permission بدە و pinch ـێکی کورت تاقی بکەرەوە.
