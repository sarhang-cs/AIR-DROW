# AIR-DROW v8.5.0 — ڕێکخستن و ڕێکخستنی وەڵامدەر بۆ کوردی

## ئەوەی کرا
- سەرەکییەک بۆ بەشی ڕێکخستن زیادکرا بۆ گەیشتن بە کامێرا و دەست، ڕووکار و پیشاندان.
- زانیاری و سیستەم/داتا بە شێوەیەکی ڕوون لە بەشی پێشکەوتوو جیاکرانەوە، بەبێ شاردنەوەی هیچ ڕێکخستنێک.
- پەیامی پاشەکەوتکردنی ناوخۆیی زیادکرا؛ هەموو گۆڕانکارییەکان لەم ئامێرە دەمێنن.
- بۆ نوسینی کوردی، line-height، قەبارەی کەمترین نووسین و ڕێگری لە دەرچوونی دەق لە مۆبایل باشتر کرا.
- کاتێک بەشی ڕێکخستن دووبارە هەڵدەبژێریت، drawer دەگەڕێتەوە سەرەوە و یەک کارتی سەرنج‌خراو دەکاتەوە.
- دڵنیابوونەوەی ڕیسێت ڕوونتر کرا: کێشان، مێژوو، پڕۆژە و گەلەری ناپارێزرێن.

## سنووری پاراستن
ئەم قۆناغە کێشان، مۆڵەتی کامێرا، MediaPipe، دەرکردن/پاشەکەوتی پارێزراو یان سنووری ڕیسێت ناگۆڕێت. ڕیسێت تەنها ڕێکخستنەکان دەگەڕێنێتەوە بۆ بنەڕەت.
