# AIR-DROW v8.7.0 — پشکنینی تەواوی بەشەکان لەسەر ئامێر

لە Settings → System & Data بەشی **پشکنینی بەشەکان** زیادکرا. ئەمە پاشەکەوتی پڕۆژە بە تۆماری کاتی و نهێنی، PNG/JPG، WebP، ڕێگای SVG/PDF، پاشەکوت/هاوبەشکردنی فایل، PWA/ئۆفلاین و سازگاریی بنەڕەتی وێبگەڕ پشکنین دەکات.

کامێرا ناکاتەوە، permission داوا ناکات، هیچ فایلێک دابەزێنێت ناکات، کێشانەکەت ناگۆڕێت و هیچ داتایەک بۆ دەرەوە نانێرێت. تۆماری پشکنینی پڕۆژە لە `finally` ـدا دەسڕدرێتەوە.

ئەمە جێگای تاقیکردنەوەی ڕاستەقینەی iPhone/Android ناگرێتەوە. پشکنینی کامێرا و دەست هەر لە بەشی Camera & Hand Check ـدایە.
