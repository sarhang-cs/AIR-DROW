# قۆناغی ٧ — پشکنینی کۆتایی لە کاتی کارکردن و Master Release

## چی زیادکرا؟

لە **Settings → دەربارەی ئەپ** پەنجەرەی **پشکنینی کۆتایی لە کاتی کارکردن** زیادکرا. ئەم پەنجەرەیە بۆخۆی کامێرا ناکاتەوە و ڕێگەپێدان داوا ناکات؛ تەنها دۆخی ڕاستەقینەی canvas، پڕۆژەی ناوخۆ، کامێرای چالاک، ئینجینی دەست و ناسینی نوێی دەست دەخوێنێتەوە.

## تاقیکردنەوەی ڕاستەقینە لە مۆبایل

1. وێبسایتەکە لە production بکەرەوە.
2. Settings → دەربارەی ئەپ → **پشکنینی ئامادەیی ئامێر** بکە.
3. بە دوگمەی سەرەکیی کامێرا، خۆت کامێرا بکەرەوە و تەنها لەو کاتەدا ڕێگەپێدان بدە.
4. یەک دەست بە ڕوونی لە چوارچێوەدا بنێ.
5. دووبارە Settings → دەربارەی ئەپ → **پشکنینی کۆتایی** بکە. دەبێت Camera، Hand Engine و Hand Detection بنووسن **ئامادە**.
6. بە pinch هێڵێکی کورت بکێشە، مشت ✊🏻 بکە، پاشان PNG export بکە و refresh بکە. دڵنیابە کێشانەکە لە ناوخۆ ماوە.

## تایبەتمەندیی پارێزگاری

پشکنینی کۆتایی هیچ `getUserMedia()` یان `enumerateDevices()` بەکارناهێنێت، هیچ داتایەک بۆ دەرەوە نانێرێت و هیچ کامێرایەک ناکاتەوە. تاقیکردنەوەی کامێرا تەنها بە دەستی خۆت و لە دوگمەی کامێرای ئەپ دەستپێدەکات.
