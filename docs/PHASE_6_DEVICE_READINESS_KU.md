# AIR-DROW v6.5.0 — قۆناغی ٦: ئامادەیی ئامێر و runtime

لە **ڕێکخستنەکان → دەربارەی ئەپ** بەشێکی نوێ هەیە بۆ پشکنینی ڕاستەقینەی ئامێر.

## چی پشکنین دەکرێت؟
- پاراستنی وێب (`HTTPS / secure context`)
- بەردەستی API ـی کامێرا، بەبێ داواکردنی ڕێگەپێدان
- گەیشتن بە مۆدێلی دەستی ناوخۆیی
- هەڵسەنگاندنی پاشەکەوتی ناوخۆ
- دۆخی PWA / Service Worker
- توانای WebGL
- دۆخی تۆڕ

## پارێزگاری
ئەم پشکنینە **هیچ کات** `getUserMedia()` بەکارناهێنێت، کامێرا ناکاتەوە، ڕێگەپێدانی کامێرا داوا ناکات، ئامێرەکان ناژمێرێت و هیچ زانیارییەک بۆ سێرڤەر نانێرێت.

ڕاپۆرتی کۆپی‌کراو تەنها ناوی دۆخەکان و ئەنجامی پشکنینە؛ نە وێنە، نە ڤیدیۆ و نە کێشانەکانت تێدا نییە.

## پشکنینی build
`npm run phase6:verify` بەشێکە لە `npm run verify:source`، و `npm run vercel:build` کۆتاییترین پشکنینی بڵاوکردنەوەیە.
