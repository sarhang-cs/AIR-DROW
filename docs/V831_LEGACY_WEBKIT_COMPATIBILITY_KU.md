# AIR-DROW v8.3.1 — سازگاری لەگەڵ WebKit ـی کۆن

## کێشەی چارەسەرکراو
هەندێک وەشانی کۆنی Safari/WebKit کۆمپايلکردنی WebAssembly وەک `unsafe-eval` حساب دەکەن، هەرچەندە `wasm-unsafe-eval` بۆ وێبگەڕە نوێکان دانرابێت. بۆیە MediaPipe ـی ناوخۆیی پێش دەستپێکردنی دۆزینەوەی دەست دەوەستا.

## چارەسەر
- CSP ـی بڵاوکراوەکە هەردوو `wasm-unsafe-eval` و `unsafe-eval` ـی سازگاری WebKit ـی کۆنی تێدایە.
- هەموو runtime، WASM و مۆدێلی دەست هەر لە ناو سەرچاوەی خۆی AIR-DROW دەخوێنرێتەوە؛ هیچ CDN ـێکی نوێ زیاد نەکراوە.
- هەڵەی درێژی ئینگلیزیی CSP/WebAssembly لە کارتی مۆبایل نیشان نادرێت؛ تەنها لە Diagnostics ـی ناوخۆیی دەپارێزرێت.
- وەشان و cache بۆ `v8.3.1 / v831` نوێکرایەوە بۆ ئەوەی دوای deploy، service worker و فایلە کۆنەکان بە پاکی بگۆڕدرێن.

## سنووری پاراستن
`unsafe-eval` تەنها بۆ سازگاری WebKit ـی کۆن زیادکراوە. AIR-DROW خۆی `eval`، `new Function`، script ـی دەرەکی یان HTML ـی بەکارهێنەر بەکارناهێنێت.
