# ڕێکخستنی Drawer و RTL — AIR-DROW v5.0.9

ئەم وەشانە بنەڕەتی ڕێکخستنی drawer ـی بەشەکان نوێ دەکات. drawer بە سێ ڕیز کار دەکات: سەردێر، tab ـەکانی workspace و ناوەڕۆکی scrollable. بۆیە بەشێک کە کراوە بێت لە ژێرەوە نابڕدرێت.

- لۆگۆی AIR-DROW و دوگمەی داخستن لە شوێنی فیزیکی جێگیرن؛ تەنها زمان و ڕاستەوخۆیی ناونیشان دەگۆڕێت.
- ئایکۆنی disclosure بە CSS geometry ـە، نەک نوسینی `‹` یان `⌃`؛ بۆیە لە کوردی لەگەڵ دەق تێک ناچێت.
- هەر card ـێک کرایەوە، drawer خۆکار بە نەرمی بۆ سەرەتای ئەو card ـە scroll دەکات.
- toggle ـەکان بە قەبارەی جێگیر و knob ـی 20px دروستکراون و لە RTL و LTR ـدا بە دروستی دەجوڵێن.


## v5.0.9 — Hand Runtime Loader Fix
ئەم وەشانە CPU-first local hand engine، drawer scroll ـی سەربەخۆ، toggle ـی تەواو، RTL/LTR geometry ـی جێگیر و redo SVG ـی لۆکاڵ زیاد دەکات.
