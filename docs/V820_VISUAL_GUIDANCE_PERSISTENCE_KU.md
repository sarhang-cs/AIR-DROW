# AIR-DROW v8.2.0 — ڕێنمایی دەست و پاشەکەوتی ڕێکخستنەکان

## ئەم وەشانە چی چاک دەکات؟

- هێڵی ڕێنمایی دەست ئێستا ٢١ خاڵ و پەیوەندییەکانی دەست لەسەر `handCanvas` پیشان دەدات. ئەم لایەیە تەنها بۆ کامێرایە و لە ناو کێشان یان فایلە دەرکراوەکان ناچێت.
- لە Settings دەتوانیت **ڕوونی هێڵی دەست** و **ئەستوری هێڵی دەست** زیاد و کەم بکەیت. هەر گۆڕانکارییەک خۆکارانە لەسەر ئامێرەکەت پاشەکەوت دەبێت.
- palette ـی ڕەنگ دوای ئەوەی کامێرا بە ڕاستی بکەوێتە کار دووبارە sync دەبێت؛ بۆیە لە Hand / Hand Eraser ـدا نابێت جارێک هەبێت و جارێک نەبێت.
- هیچ هێڵی دەست، palette یان UI ـیەک لە PNG/JPG export ـدا ناچێت. Download، Save، Export، Share و Clear تەنها بە کرتەی ڕاستەقینەی بەکارهێنەر کاردەکەن.
- کارتی Instagram و تایپۆگرافی کوردی بۆ شاشەی بچووک کورتتر و ڕێکخراوتر کراون.

## ڕیسێتی ڕێکخستنەکان

Reset Settings تەنها لە دوای پشتڕاستکردنەوە ڕێکخستنەکان دەگەڕێنێتەوە بۆ بنەڕەت. کێشانەکان، ناوی project، gallery و view ناسڕدرێنەوە.

## چێکی مۆبایل

دوای deploy: Hand Mode بکەرەوە، مۆڵەتی کامێرا بدە، دڵنیابە palette دەرکەوێت، opacity/thickness بگۆڕە، ئەپەکە دابخە و بیکەرەوە تا پاشەکەوتبوونیان ببینیت. PNG/JPG export بکە و دڵنیابە تەنها artwork دەردەچێت.
