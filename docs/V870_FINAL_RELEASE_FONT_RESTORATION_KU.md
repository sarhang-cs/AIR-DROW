# AIR-DROW v8.7.0 — ئامادەیی بڵاوکردنەوەی کۆتایی و گەڕاندنەوەی فۆنت

## ئەوەی ئەنجام درا
- فۆنتی کوردیی ناوخۆیی گەڕێندرایەوە بۆ سەرچاوەی ئەپ و بە build دەگوازرێتەوە بۆ وەشانی بڵاوکراوە.
- `@font-face` و preload ـی هەمان فۆنت بە یەک ڕێگا کار دەکەن.
- header ـی ڕاستی فۆنت و پشکنینی سەرچاوە/build زیادکرا.
- هەموو چاکسازییەکانی پێشوو بۆ دەست، export، safety، WebKit و Settings پارێزراون.

## سنووری ڕاستی بڵاوکردنەوەی کۆتایی
QA ـی ئۆتۆماتیک کۆد، build، فایلە ناوخۆییەکان و ڕێگای export دەچێکێت؛ بەڵام شوێنی تاقیکردنەوەی ڕاستەقینەی Android Chrome و iPhone Safari ناگرێتەوە. پێش GitHub Release ـی کۆتایی، permission ـی کامێرا، Hand Tracking، PNG/JPG download، PWA install و reopen/update لەسەر مۆبایلی ڕاستەقینە تاقی بکەوە.

## دڵنیابوونەوەی فۆنت
دوای deploy، زمان بگۆڕە بۆ کوردی و Settings یەکجار دابخە و بکەرەوە. نوسینە کوردییەکان دەبێت بە فۆنتی ناوخۆیی و بێ گۆڕانی ناڕێک دەرکەون.
