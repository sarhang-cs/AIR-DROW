# AIR-DROW v6.1.0 — قۆناغی ٢: ڕووکار، RTL و مۆبایل

## ئەوەی چاک کرا
- toolbar ـی مۆبایل دوگمەکان بچووک ناکات و نایانشارێتەوە؛ لە شاشەی باریکدا بە ئارامی scroll دەکرێت.
- Settings لە مۆبایل bottom-sheet ـی safe-area ـە و لە desktop side-workspace ـێکی پاکە.
- دۆخی Dark و Light لە Settings بە ڕاستەوخۆ هەڵدەبژێردرێت؛ دوگمەی سەرەوە تەنها quick toggle ـە.
- شەش ستایلی تەواوی UI هەن: Violet، Pink، Sapphire، Obsidian، Silver و Gold.
- RTL/LTR بۆ ناونیشان، dropdown، toggle، arrow و logo بە جیاوازی بە سەلامەتی ڕێکخرا.
- FPS، Online و CAM هەر بە دەق ماونەتەوە و هیچ background/card ـێکیان نییە.

## چێکی مۆبایل
1. Settings بکەرەوە و تا کۆتایی scroll بکە؛ هیچ control ـێک نابێت لەژێر bottom navigation بشاردرێتەوە.
2. Dark/Light و هەموو شەش skin ـەکە بگۆڕە و refresh بکە؛ هەڵبژاردەکە دەبێت بمێنێت.
3. زمان بگۆڕە بۆ Kurdish و English؛ label و arrow و dropdown نابێت یەکدی بگرن.
4. کامێرا بکەرەوە؛ دڵنیابە FPS/Online/CAM بەبێ panel دەرکەون.

## Build
لە repo: `npm run vercel:build`
