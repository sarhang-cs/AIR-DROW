# AIR-DROW — چێکلیستی پشکنینی ڕاستەقینەی Android

ئەم چێکلیستە **دوای deploy ـی هەمان وەشان** بەکاربهێنە. بۆ پشکنینی مۆبایلی ڕاستەقینەیە، نەک تەنها ڕێکلام. لە Android Chrome ـەوە URL ـی Vercel ـەکە بکەرەوە.

## پێش کامێرا

- [ ] URL ـی deploy کراو بکەرەوە و جارێک refresh بکە. لە **ڕێکخستن → دەربارەی ئەپ** دڵنیابە وەشانی دیارکراو هەمان وەشانی ئێستایە.
- [ ] لە **ڕێکخستن → سیستەم → پشکنینی ئەپ** پشکنین بکە. secure context، camera API، model ـی ناوخۆیی، storage، PWA، graphics، network، شوێنی کێشانی کاتی، PNG/JPG و ئامادەکردنی فایل دەبێت دیار بن. پشکنینەکە نابێت کامێرا بکاتەوە یان فایل دابەزێنێت.
- [ ] toolbar، bottom dock، tab ـەکانی Settings، شوێنی palette ـی سەرەوە و پشت modal دابگرە. هیچ یەکێکیان نابێت هێڵی کێشان دروست بکات.
- [ ] یەک هێڵ بە تەچ بکێشە، Undo و Redo بکە و دڵنیابە هێڵەکە تەنها لە ناو canvas ـدایە.

## کامێرا و دەست

- [ ] **کردنەوەی کامێرا** هەڵبژێرە. permission ـی کامێرا تەنها ئێرە دەبێت داوا بکرێت، نەک لە کاتی یەکەم کردنەوەی ئەپ.
- [ ] permission بدە. چاوەڕێ بکە تا engine ـی ناوخۆیی ئامادە بێت، پاشان یەک دەستت لە چوارچێوە پیشان بدە.
- [ ] بە index/thumb ـی ڕێکخراو بۆ ٢٠ چرکە بکێشە. دەستت بۆ ماوەیەکی کورت دەرخە و بگەڕێنەوە. هێڵی درێژی بازدراو نابێت دروست بێت.
- [ ] دەستی track کراوت بەسەر toolbar، dock، palette و Settings ـدا ببرە. هیچ button، download، clear، export یان گۆڕینی tab نابێت بە دەست trigger بکرێت.
- [ ] gesture shortcuts تەنها ئەگەر خۆت دەتەوێت چالاک بکە. دوو پەنجە ڕابگرە و دڵنیابە تەنها **Hand Eraser** دەگۆڕێت. هیچ file action ـێک نابێت ڕووبدات.
- [ ] index بەسەر ڕەنگێکی palette ـەوە بگرە تا ماوەی پێویست بڕوات. تەنها دوای dwell ڕەنگەکە بگۆڕێت و لە ژێر palette هێڵ دروست نەبێت.

## Export و Offline

- [ ] هێڵێکی دیار بکێشە. بە پەنجەی ڕاستەقینە **Save/Export** دابگرە و PNG و JPG دەرکە.
- [ ] هەردوو فایلەکە بکەرەوە. دەبێت artwork ـەکە تێدا بێت، نەک تەنها background. UI، camera feed، hand guide و skeleton نابێت لە export ـدا دەربکەون.
- [ ] دوای یەکەم بارکردن، ئینتەرنێت کوژێنەوە، جارێک refresh بکە و ئەپەکە بکەرەوە. ستۆدیۆی کێشان دەبێت هەر بارببێت؛ hand model تەنها کاتێک کامێرا هەڵدەبژێریت داوا دەکرێت و دەتوانێت پاشەکەوت بکرێت.
- [ ] ئینتەرنێت بگەڕێنەوە، update deploy بکە، URL بکەرەوە و دڵنیابە update banner یان version ـەکە بە پاکی نوێ دەبێتەوە، بەبێ black screen یان loading ـی بێکۆتایی.

## ڕاپۆرت

لە **ڕێکخستن → سیستەم → پشکنینی ئەپ** و **پشکنینی کامێرا و دەست** ڕاپۆرتی ناوخۆیی کۆپی بکە. ڕاپۆرتەکە بۆ debugging ـە و artwork، camera frame، landmark، ناوی پڕۆژە و API URL تێدا ناگرێت.

تەنها کاتێک release ـەکە بۆ GitHub tag بکە کە هەموو خاڵە پەیوەندیدارەکان لەسەر ئەو مۆبایلەی ئامانج پاس بوون.
