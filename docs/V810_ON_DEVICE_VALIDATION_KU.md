# AIR-DROW v8.1.0 — پشکنینی ڕاستەقینەی ئامێر پێش بڵاوکردنەوە

## ئامانج

ئەم قۆناغە بۆ ئەوەیە کۆتایی release ـەکە بە ڕاستی و بە پاراستنی تایبەتمەندی بپشکنرێت. تاقیکردنەوەی کۆد و build دەتوانێت فایل و لۆجیک بپشکنێت، بەڵام ناتوانێت وەک مۆبایلێکی ڕاستەقینە permission ـی کامێرا یان download sheet ـی وێبگەڕ بکاتەوە.

## ئەوەی لە ڕێکخستن → دەربارەی ئەپ → پشکنینی ئەپ زیادکرا

پشکنینی ناوخۆیی ئەمانە دەپشکنێت، بەبێ کردنەوەی کامێرا، بەبێ دابەزاندنی فایل، بەبێ خوێندنەوەی artwork ـی تۆ و بەبێ ناردنی ڕاپۆرت بۆ دەرەوە:

- secure context و بوونی Camera API.
- بەردەستبوونی model ـی دەستی ناوخۆیی و same-origin.
- storage، PWA، graphics و پەیوەندی ناوخۆیی بە release.json.
- canvas ـێکی کاتی 16 × 16 بتوانێت pixel بکێشێت.
- هەمان canvas ـی کاتی بتوانێت PNG و JPG دروست بکات.
- وێبگەڕ بتوانێت local object URL بۆ فایل ئامادە بکات و پاشان بسڕێتەوە.

ئەم canvas ـە کاتییە یەکسەر دەسڕدرێتەوە. هیچ pixel ـی پڕۆژە، frame ـی کامێرا، landmark ـی دەست یان metadata ـی تایبەت بەکارناهێنرێت.

## سنووری ڕوون

`ئامادەکردنی فایل` تەنها دڵنیایی دەداتەوە AIR-DROW دەتوانێت فایل ئامادە بکات. بەڵام نایڵێت Android download sheet کراوەتەوە. بۆ ئەوە، هێڵێکی دیار بکێشە و بە دوگمەی ئاسایی Export، PNG یان JPG بە دەستی خۆت دەرکە.

هەروەها `کامێرا` تەنها بوونی interface ـی کامێرا دەپشکنێت. بۆ پشکنینی ڕاستەقینەی کامێرا و hand tracking، خۆت کامێرا بکەرەوە، permission بدە، دەستت پیشان بدە و **پشکنینی کێشان بە دەست** دووبارە بکە.

## دەروازەکانی ئۆتۆماتیک

- `npm run qa:launch-validation`
- `npm test`
- `npm run verify:all`

## دەروازەی release

GitHub Release tag تەنها دوای ئەوە دروست بکە کە build و خاڵە پەیوەندیدارەکانی `docs/DEVICE_VALIDATION_CHECKLIST_KU.md` لەسەر مۆبایلی ئامانج بە سەرکەوتوویی تەواو بوون.
