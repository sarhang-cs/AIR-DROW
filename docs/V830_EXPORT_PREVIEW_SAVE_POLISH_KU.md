# AIR-DROW v8.3.0 — پێشبینینی دەرکردن و پاکسازیی پاشەکەوت

## چی نوێکراوەتەوە

- دوگمەی **پێشبینینی دەرکردن** زیادکرا. پێش دروستکردنی فایل، دیمەنێکی ناوخۆیی دروست دەکات بە هەمان Fit / Fill / Stretch، شفافیت، کامێرا و شێوازی brush ـی فایلە کۆتاییەکە.
- پێشبینین هیچ کات download ناکات. فایل تەنها دوای تاپی ڕاستەقینەی بەکارهێنەر لە **دەرکردن** یان **بڵاوکردنەوە** دروست دەبێت.
- پەنجەرەی مۆبایلی پارێزراو زیادکرا، بە دوگمەی ڕوونی «ئێستا دەر بکە» و «بڵاوکردنەوە». دەست و کامێرا ناتوانن لە پشت پەنجەرەکە بنووسن یان دوگمەیەک چالاک بکەن.
- URL ـی کاتیی پێشبینین کاتێک دایدەخەیت یان پێشبینینی نوێ دروست دەکەیت پاک دەکرێتەوە؛ بۆیە بیرگەی مۆبایل بەخۆڕایی پڕ نابێت.
- QA ـی نوێ زیادکرا کە دڵنیابێت preview هیچ بانگێکی download تێدا نییە.

## پشکنین

پێش release: `npm test`، `npm run qa:export-preview-save`، `npm run preflight` و `npm run check`.
