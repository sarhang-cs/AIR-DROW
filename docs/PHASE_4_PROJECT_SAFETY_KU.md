# AIR-DROW v6.3.0 — قۆناغی ٤: پاراستنی پڕۆژە، دەرکردن و ڕێنمایی سەرەتایی

ئەم قۆناغە هەموو کارەکان لەسەر ئامێرەکەت دەپارێزێت و هیچ داتای کێشان بەخۆکار بۆ سێرڤەر نانێرێت.

- کاتێک هێڵێک دەکێشیت، پێش ئەوەی پاشەکەوتی IndexedDB تەواو بێت، draft ـێکی کورت و سنووردار لە هەمان session ـدا دەپارێزرێت.
- ئەگەر وێبگەڕ نوێبکرێتەوە، tab ـەکە بخەوێت یان لە پاشبنەما وەستێت، تەنها کاتێک ئەو draft ـە دەگەڕێتەوە کە لە پاشەکەوتی بنەڕەت نوێتر بێت.
- دەرکردن هەمیشە یەکەم جار پڕۆژەکەت پاشەکەوت دەکات. ئەگەر WebP لە وێبگەڕەکەت نەکرایەوە، بێ هەڵە PNG دەرئەکات.
- باکئەپ لە مۆبایلی Android بۆ ماوەی پێویست URL ـی فایلەکە دەپارێزێت تا داگرتنەکە نەپچڕێت.
- نوێکردنەوەی ئەپ پێش گواستنەوە پاشەکەوتکردن دەسەلمێنێت؛ ئەگەر پاشەکەوت سەرکەوتوو نەبێت، نوێکردنەوە جێبەجێ ناکرێت.
- ڕێنمایی سەرەتایی بە swipe، تیرەکانی کیبۆرد و گەڕاندنەوەی focus کاردەکات.
