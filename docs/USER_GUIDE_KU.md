# ڕێنمایی بەکارهێنانی AIR-DROW

> ئەم ڕێنماییە بۆ AIR-DROW 8.4.0 — جێگیریی کێشانی بە دەستە.

## دەستپێکردن بە کێشان

1. AIR-DROW بکەرەوە.
2. لە toolbar ـی سەرەوە فڕچە یان پاککەرەوە هەڵبژێرە.
3. بە تەچ، ماوس یان قەڵەم لەسەر بۆرد بکێشە.
4. کاتێک پێویست بوو Undo و Redo بەکاربهێنە.

## کێشان بە دەست

1. **دەست** یان **کامێرا** هەڵبژێرە.
2. پەیامی کامێرا بخوێنەوە و تەنها کاتێک ئامادەیت ڕێگەپێدان بدە.
3. یەک دەستت بە تەواوی لە شوێنێکی ڕوون لە چوارچێوەدا بگرە.
4. پەنجەکانت لەیەک نزیک بکە، پاشان دەستت بجوڵێنە بۆ کێشان.
5. دوو پەنجەت بۆ ماوەیەکی کورت ڕابگرە بۆ گۆڕینی **پاککەرەوەی دەست**، پاشان pinch بکە بۆ سڕینەوە.
6. پەنجەی نیشانە بە سەر پالێتی ڕەنگی سەرەوەدا بگرە بۆ گۆڕینی ڕەنگ.

AIR-DROW تا خۆت دوگمەی کامێرا هەڵنەبژێریت، کامێرا ناکاتەوە.

## پاشەکوتی پڕۆژە

بەشی پڕۆژەکان بکەرەوە، ئەگەر دەتەوێ ناوی پڕۆژەکە بنووسە، پاشان **پاشەکوت** بکە. پڕۆژەکەت لە هەمان وێبگەڕدا دەپارێزرێت. پێش گواستنەوە بۆ ئامێرێکی نوێ backup دابەزێنە.

## پێشبینینی دەرکردن و دەرکردنی کار

1. لە ڕێکخستنی Draw، PNG، JPG، WebP، SVG، PDF یان پڕۆژەی AIR-DROW هەڵبژێرە.
2. قیاس، کوالێتی و Fit / Fill / Stretch دیاری بکە.
3. **پێشبینینی دەرکردن** تاپ بکە بۆ بینینی ئەنجام؛ لەم هەنگاوەدا هیچ فایلێک دابەزێنرێت ناکات.
4. تەنها کاتێک دڵنیایت، **ئێستا دەر بکە** یان **بڵاوکردنەوە** هەڵبژێرە.

AIR-DROW پێش دەرکردنی واقعی پڕۆژەکەت پاشەکەوت دەکات. تێکەڵکردنی کامێرا تەنها بە هەڵبژاردنی تۆ کاردەکات؛ UI و هێڵی ڕێنمایی دەست هەرگیز لە فایلدا نایەن.

## تایبەتمەندی

کێشان، ڕێکخستن و diagnostics ـی ناوخۆیی لەسەر ئامێرەکەت دەمێنن، مەگەر خۆت دەرکردن، بڵاوکردنەوە یان دروستکردنی AI هەڵبژێریت. AIR-DROW frame ـی کامێرا پاشەکەوت ناکات.
