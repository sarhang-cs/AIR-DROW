# ڕێنمایی بەکارهێنانی AIR-DROW

> ئەم ڕێنماییە بۆ AIR-DROW 7.7.1 Critical Input & Export Recovery Edition

## دەستپێکردن بە کێشان

1. AIR-DROW بکەرەوە.
2. لە toolbar ـی سەرەوە فڕچە یان پاککەرەوە هەڵبژێرە.
3. بە تەچ، ماوس یان قەڵەم لەسەر بۆرد بکێشە.
4. کاتێک پێویست بوو Undo و Redo بەکاربهێنە.

## کێشان بە دەست

1. **دەست** یان **کامێرا** هەڵبژێرە.
2. پەیامی کامێرا بخوێنەوە و تەنها کاتێک ئامادەیت ڕێگەپێدان بدە.
3. یەک دەستت بە تەواوی لە شوێنێکی ڕوون لە چوارچێوەدا بگرە.
4. پەنجەکانت لەیەک نزیک بکە، پاشان دەستت بجوڵێنە بۆ کێشان.
5. دوو پەنجەت بۆ ماوەیەکی کورت ڕابگرە بۆ گۆڕینی **پاککەرەوەی دەست**، پاشان pinch بکە بۆ سڕینەوە.
6. پەنجەی نیشانە بە سەر پالێتی ڕەنگی سەرەوەدا بگرە بۆ گۆڕینی ڕەنگ. ئەگەر tracking ون بوو، هێڵ بە سەلامەتی دەوەستێت یان دەپارێزرێت.

AIR-DROW تا خۆت دوگمەی کامێرا هەڵنەبژێریت، کامێرا ناکاتەوە.

## پاشەکەوتی پڕۆژە

بەشی پڕۆژەکان بکەرەوە، ئەگەر دەتەوێ ناوی پڕۆژەکە بنووسە، پاشان **پاشەکەوت** بکە. پڕۆژەکەت لە هەمان وێبگەڕدا دەپارێزرێت. پێش گواستنەوە بۆ ئامێرێکی نوێ backup دابەزێنە.

## دەرکردنی کار

بەشی Export بکەرەوە و PNG، JPG، WebP، SVG، PDF یان پڕۆژەی AIR-DROW هەڵبژێرە. بۆ وێنە، کوالێتی 1×، 2× یان 4×، ڕێکخستنی Fit/Fill/Stretch و قیاسی سۆشیال یان پرینت دیاری بکە. AIR-DROW یەکەم جار پڕۆژەکە پاشەکەوت دەکات. تێکەڵکردنی کامێرا تەنها بە هەڵبژاردنی تۆ کاردەکات و UI هەرگیز لە فایلدا نایەت.

## پشکنینی ئەپ

کاتێک دەتەوێ دڵنیابیت کە پاشەکەوت، گرافیک و فایلەکانی AIR-DROW ئامادەن، بچۆ **ڕێکخستنەکان → دەربارەی ئەپ → پشکنینی ئەپ**.

## پشکنینی کامێرا و دەست

سەرەتا خۆت کامێرا بکەرەوە. کاتێک وێنەی زیندوو دەرکەوت، بچۆ **ڕێکخستنەکان → دەربارەی ئەپ → پشکنینی کامێرا و دەست** بۆ دڵنیابوون لە کێشان بە دەست.

## تایبەتمەندی

کێشان و ڕێکخستنەکانت ناوخۆیین، مەگەر خۆت export، share یان دروستکردنی AI هەڵبژێریت. ڤیدیۆی کامێرا لەلایەن AIR-DROW ـەوە پاشەکەوت ناکرێت.


> **v7.7.1:** دەرکردنی کوالێتی بەرز، gesture ـی پالێت و سڕینەوە، smoothing ـی خۆکار و diagnostics ـی تایبەتی زیادکرا.
