AIR-DROW v7.5.1 — one-command safe replacement

Download AIR-DROW_v7.5.1_BOOTSTRAP_HOTFIX.zip to /sdcard/Download, then paste this into Termux. Change only REPO if your old GitHub project is in a different folder:

REPO="$HOME/AIR-DROW-GITHUB"; ZIP="/sdcard/Download/AIR-DROW_v7.5.1_BOOTSTRAP_HOTFIX.zip"; pkg update -y && pkg install -y nodejs-lts git unzip && rm -rf "$HOME/AIR-DROW-V751" && mkdir -p "$HOME/AIR-DROW-V751" && unzip -q "$ZIP" -d "$HOME/AIR-DROW-V751" && bash "$HOME/AIR-DROW-V751/AIR-DROW-v7.5.1-BOOTSTRAP-HOTFIX/termux/replace-with-v751-bootstrap-hotfix.sh" "$REPO"

This backs up all old project files except .git, installs exact dependencies, builds, verifies, commits and pushes. If verification fails, it restores the old project automatically.
