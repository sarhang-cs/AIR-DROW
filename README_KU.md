## AIR-DROW چی دەکات؟

AIR-DROW ئەپێکی PWA ـی وەڵامدەرە بۆ کێشان بە تەچ، قەڵەم و ئارەزوومەندانە بە دەست. پڕۆژە، ڕێکخستن، کالیبراسیۆن و دەرکردنەکانت بە بنەڕەت لەسەر ئامێرەکەت دەمێنن. ناساندنی دەست بە MediaPipe ـی ناوخۆیی کاردەکات و کامێرا هەرگیز بەبێ هەڵبژاردنی تۆ ناکرێتەوە.

- بە تەچ، قەڵەم، بڕۆش، سڕینەوە و یارمەتی شێوەکان بکێشە.
- تەنها کاتێک خۆت هەڵبژێریت، کامێرا بۆ کێشان بە دەست بکەرەوە.
- پڕۆژەکانت لەسەر ئامێر پاشەکەوت بکە، PNG/JPG/WebP/SVG/PDF دەر بکە و باکئەپ دروست بکە.
- هەموو ڕووکارەکە بە تەواوی لە نێوان **کوردی (سۆرانی)** و **ئینگلیزی** بگۆڕە.
- لە مۆر، پینک، سافایەر، ئۆبسیدیان، سلڤەر و گۆڵد هەڵبژێرە؛ هەریەکەیان دۆخی تاریک و ڕوونی تایبەتی خۆی هەیە.

## شێوازەکانی کێشان

**تەچ و قەڵەم:** AIR-DROW بکەرەوە و یەکسەر دەست بە کێشان بکە. قەبارەی هێڵ، نەرمی، ڕەنگ، فشار، undo و redo لە ستودیۆکەدا بەردەستن.

**کێشان بە دەست:** کامێرا بکەرەوە، چاوەڕێ بکە تا engine ـی دەست ئامادە ببێت، پاشان یەک دەستت لە چوارچێوە پیشان بدە. index و thumb لەیەک نزیک بکە بۆ کێشان. index بە سەر پالێتی سەرەوەدا بگرە بۆ ڕەنگ؛ دوو پەنجە ڕابگرە بۆ Hand Eraser. ئەگەر دەستت دەرچوو یان tracking ناجێگیر بوو، هێڵ بە سەلامەتی وەستێنرێت.

**یارمەتی شێوەکان:** بۆ هێڵ، بازنە، چوارگۆشە و سێگۆشەی ڕێک یارمەتی شێوەکان بەکاربهێنە، بەبێ لەدەستدانی کۆنترۆڵی تەواو.

## تایبەتمەندی

- کێشان، پڕۆژە، ڕێکخستن و کالیبراسیۆن لەسەر ئامێرەکەت دەمێنن.
- ڕێگەپێدانی کامێرا تەنها دوای هەڵبژاردنی **کردنەوەی کامێرا** داوا دەکرێت.
- فریمەکانی کامێرا تەنها بۆ ناساندنی دەست بەکاردێن و تۆمار ناکرێن.
- model و runtime ـی دەست لە خۆی AIR-DROW ـەوە دێن، نەک لە CDN ـی کەسێکی تر.
- پشکنین و diagnostics لەسەر ئامێرەکەت کاردەکەن و ڕاپۆرت بۆ شوێنێکی تر نانێرن؛ frame ـی کامێرا، landmark، خاڵی کێشان، ناوی پڕۆژە و API URL تێدا نایەن.

[ڕێنمایی تەواوی تایبەتمەندی](./docs/PRIVACY.md) بخوێنەوە.

## زمان و theme

لە **ڕێکخستن → دەربارەی ئەپ** زمان لە کوردی بۆ ئینگلیزی یان بە پێچەوانە بگۆڕە. کارتە داینامیکییەکان، کالیبراسیۆن، پەیامی کامێرا و پەیامی چارەسەرکردنەوە یەکسەر بە زمانی هەڵبژێردراو نوێ دەبنەوە.

لە **ڕێکخستن → دیزاین** دۆخی تاریک/ڕوون و یەکێک لە شەش theme ـەکە هەڵبژێرە. هەموو theme ـەکان contrastی ڕاستیان بۆ دەق، دوگمە و ئایکۆن هەیە.

## Build و deploy

```bash
npm ci --no-audit --no-fund
npm run vercel:build
```

بۆ گۆڕینی پڕۆژەکە لە Android/Termux، [`termux/TERMUX_INSTALL.txt`](./termux/TERMUX_INSTALL.txt) بەکاربهێنە. script ـەکە `.git` دەپارێزێت، backup دروست دەکات، build دەکات، و تەنها دوای سەرکەوتن commit/push دەکات.

پێش دانانی tag ـی کۆتایی لە GitHub، [چێکلیستی پشکنینی مۆبایل](./docs/DEVICE_VALIDATION_CHECKLIST_KU.md) تەواو بکە. ئەمە لە تاقیکردنەوەی ئۆتۆماتیکی جیاوازە، چونکە تەنها مۆبایلی ڕاستەقینە دەتوانێت camera driver، permission و download sheet پشتڕاست بکاتەوە.

## گرنگترین نوێکارییەکانی v8.0.0

- **Export Studio:** PNG، JPG و WebP بە کوالێتی 1×، 2× و 4×؛ شێوازی Fit/Fill/Stretch، قیاسی سۆشیال و A4/A3 ـی 300 PPI.
- **کۆنترۆڵی دەست:** پەنجەی نیشانە بە سەر پالێتی سەرەوەدا بگرە بۆ گۆڕینی ڕەنگ؛ دوو پەنجە ڕابگرە بۆ گۆڕینی Hand Eraser. هیچ gesture ـێکی دەست ناتوانێت Save، Export، Clear، Share یان Download دەستپێبکات؛ هەموویان تەنها بە کرتەی ڕاستەقینەی بەکارهێنەر کاردەکەن.
- **کێشانی جێگیر:** smoothing ـی خۆکار لەرزە کەم دەکات. کاتێک دەست لە چوارچێوە دەچێتە دەرەوە، هێڵ بە سەلامەتی وەستێنرێت یان دەپارێزرێت، نەک ئەوەی بازبدات.
- **ڕاپۆرتی تایبەت:** هەڵەی کامێرا، دەست و export دەتوانرێت تەنها لە ناو ئامێرەکەت کۆپی یان دابەزێنرێت؛ هیچ frame، landmark، خاڵی کێشان، ناوی پڕۆژە یان API URL تێدا نییە.
