# AIR-DROW v5.2.3 — چاکسازیی بەردەوامی ڕێنمایی مشت

کاتێک دەست دەکرێت بە مشت ✊🏻، ئەگەر MediaPipe landmark ـەکانی دەست بدۆزێتەوە، هێڵ و خاڵی ڕێنماییی دەست دیار دەمێنن.

ئەم گۆڕانکارییە تەنها بۆ پیشاندانی guide ـە. یاساکانی ئاسایشی drawing و pinch هەر بەهێزن؛ واتە مشت نەبێتە هۆی دەستپێکردنی هێڵکێشان.

هەروەها کاتێک لەبەر داخستنی پەنجەکان detector بۆ یەک frame ون ببێت، guide ـەکە بۆ ماوەیەکی زۆر کورت دەگیرێت تا پەپول/ونبوونی ناگهانی نەبێت.

`FPS`، `Online` و `CAM` هەر تەنها دەق و خاڵی دۆخن؛ هیچ باکگراوەند، box، border، pill یان blur ـیان نییە.

## Deploy
```bash
npm install
npm run build
```

دوای push بۆ GitHub، Vercel خۆکارانە deploy دەکات.

## v5.2.3 — ڕێکخستنی پشکنینی وەشان

پشکنینی کۆتایی ئێستا HUD ـی شەفاف و بەردەوامی ڕێنمایی مشت بە جیاوازی پشکنین دەکات، بۆیە هەڵەی ساختەی build دووبارە نابێت.
