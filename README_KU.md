# AIR-DROW v7.5.6 — وەشانی داخڵکردنی نووسین، زمان و دەست

ئەم وەشانە ڕێکخستنی کۆتایی بۆ داخڵکردنی دەق لە مۆبایل، گۆڕینی زمان، بارکردنی دەست و هەموو زیمەکانە.

- کاتێک زمان دەگۆڕیت بۆ **کوردی**، هەموو دەقی داینامیکی — کالیبراسیۆن، پشکنینی دەست، کارتی ئامادەیی، پەیامەکانی کامێرا و دۆخی تۆڕ — کوردی دووبارە نیشان دەدرێت.
- کاتێک زمان دەگۆڕیت بۆ **English**، هەمان بەشەکان بە تەواوی ئینگلیزی دەبن؛ هیچ دەقی کوردی لە دۆخی پێشوەوە نامێنێت.
- هەردوو dictionary ـەکە یەک ژمارە keyی verifiedیان هەیە؛ fallback بۆ زمانی دیکە بەکارناهێنرێت.
- شەش زیمی تایبەت هەیە: Violet، Pink، Sapphire، Obsidian، Silver و Gold. هەر یەکەیان دارک و لایتی تایبەتی خۆی هەیە.
- ڕەنگی ئایکۆنەکان بە tokenی contrast پارێزراوە؛ لە دارک و لایتدا ون نابن، و دۆخی چالاک بە ڕەنگی گونجاو دیارە.
- قوفڵی long-press و selection لە هەموو ڕووکاری ئەپ — بە onboarding و ڕێنمایییەکانیشەوە — چالاکە؛ تەنها input/textarea/selectی ڕاستەقینە بۆ نووسین بەکار دەهێنرێن.
- خانەکانی نووسین لە Android بە یەکەم داگرتن focus دەبن و کیبۆرد دەکەنەوە.
- لە یەکەم کردنەوەی کامێرا، ئەپ تەنها دوای ئامادەبوونی runtime و model دەڵێت دەستت لەبەردەم کامێرا ڕابگرە.
- دوای تەواوبوونی کالیبراسیۆن، پەیامی سەرکەوتن و پاشەکەوتکراوی پیشان دەدرێت، کەمێک دەمێنێتەوە و بە animation لادەچێت.

## بۆ build
```bash
npm ci --no-audit --no-fund
npm run vercel:build
```

## بۆ Termux
فایلی [`termux/TERMUX_INSTALL.txt`](./termux/TERMUX_INSTALL.txt) بکەرەوە؛ یەک کۆدی کامل بۆ جێگۆڕکردنی پڕۆژەی کۆن، backup و پشکنینی buildی نوێ تێدایە.
