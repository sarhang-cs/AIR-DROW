## بەخێربێیت بۆ AIR-DROW

**AIR-DROW** ستودیۆیەکی تایبەتیی کێشانە بۆ ئەو بیرۆکانەی لە هەر شوێنێک دەست پێدەکەن. بە پەنجە یان قەڵەم ڕاستەوخۆ بکێشە، یان کاتێک دەتەوێت، کامێرا بکەرەوە و بە دەستت لە هەوا بکێشە. پڕۆژەکانت پاشەکەوت بکە، backup دروست بکە، و کارە هونەرییەکەت دەرکە یان بڵاوی بکەوە.

بۆ **مۆبایل، تابلێت و کۆمپیوتەر** دروست کراوە و بە **کوردی سۆرانی** و **English** بەردەستە. سەرەتا بکێشە؛ ڕێکخستنەکان تەنها کاتێک بەکاربهێنە کە پێویستت پێیانە.

## بە کورتی

| دروست بکە | پارێزە | بڵاوی بکەوە |
| --- | --- | --- |
| بە تەچ، قەڵەم، فڕچە، پاککەرەوە و شێوەی زیرەک بکێشە | پڕۆژەکانت لەسەر ئامێر پاشەکەوت بکە و backup دابەزێنە | وێنەکانت دەرکە و لە ڕێگەی ئامێرەکەتەوە بڵاویان بکەوە |
| بە هەڵبژاردەی کامێرا و دەست لە هەوا بکێشە | draft ـی نوێت بە دڵنیایی بگەڕێنەوە | پۆستەر، کارت، ڕیپلەی و template دروست بکە |

## چۆن بکێشیت

### تەچ و قەڵەم

تەنها AIR-DROW بکەرەوە و دەست بە کێشان بکە. فڕچە هەڵبژێرە، ڕەنگ و قەبارە دیاری بکە، و هەر کات پێویست بوو Undo یان Redo بەکاربهێنە.

### کێشان بە دەست

دوگمەی **دەست** یان **کامێرا** هەڵبژێرە، کاتێک داوات لێکرا ڕێگەپێدانی کامێرا بدە، پاشان یەک دەست لە چوارچێوەدا بگرە. پەنجەکانت لەیەک نزیک بکە و بجوڵێ بۆ کێشان. AIR-DROW خۆی کامێرا ناکاتەوە؛ تۆ هەمیشە دەستپێکەرەکەیت.

### یارمەتیی شێوەکان

کاتێک دەتەوێت هێڵ، بازنە، چوارگۆشە یان سێگۆشە ڕێک‌تر بێت، یارمەتیی شێوەکان چالاک بکە. هێشتا هەموو کنترۆڵ لەدەست تۆدایە.

## پاشەکەوت، دەرکردن و بڵاوکردنەوە

- **پاشەکەوتی پڕۆژە** بۆ ئەوەی کارەکانت لە هەمان وێبگەڕدا بمێنن.
- **Backup** بکە پێش گۆڕینی مۆبایل، تابلێت یان وێبگەڕ.
- **Export** بکە بۆ دەرکردنی کارە هونەرییەکانت.
- **Share** بکە لە ڕێگەی تایبەتمەندییەکانی ئامێرەکەتەوە.
- **Template** دروست بکە بۆ پۆستەر، لۆگۆ، ستۆری و بیرۆکەکانی پرێنت.

پێش دروستبوونی هەر export ـێک، پڕۆژەی ئێستات پاشەکەوت دەکرێت. ئەگەر فۆرماتێک لە وێبگەڕەکەت پشتیوانی نەکرێت، AIR-DROW هەڵبژاردەیەکی ناوخۆیی و گونجاو پێشنیار دەکات تا کارت لەدەست نەدەیت.

## تایبەتمەندیی یەکەمە

- کێشان، پڕۆژە و ڕێکخستنەکانت بە بنەڕەت لەسەر ئامێرەکەت دەمێنن.
- کامێرا تەنها دوای ئەوەی تۆ دوگمەی کامێرا هەڵبژێریت ڕێگەپێدان داوا دەکات.
- ڤیدیۆی کامێرا لەلایەن AIR-DROW ـەوە پاشەکەوت ناکرێت.
- مۆدێل و ئینجینی دەست لەناو ئەپەکەدان؛ لە کاتی کێشان مۆدێلێکی دەست لە دەرەوە دابەزێنرێت نا.
- پشکنینی ئەپ و پشکنینی کامێرا و دەست تەنها لەسەر ئامێرەکەت کاردەکەن و هیچ ڕاپۆرتێک نانێرن.
- دروستکردنی AI هەمیشە بە هەڵبژاردنی خۆتە؛ تەنها ئەو sketch ـەی خۆت بۆی دیاری دەکەیت دەنێردرێت بۆ خزمەتگوزاری AI ـی ڕێکخراو.

وردەکاری زیاتر لە [ڕێنمایی تایبەتمەندی](./docs/PRIVACY.md) بخوێنەوە.

## پشکنینی ئەپ و پشکنینی کامێرا و دەست

لە **ڕێکخستنەکان → دەربارەی ئەپ** دوو یارمەتیدەر هەیە:

- **پشکنینی ئەپ** دڵنیات دەکاتەوە کە AIR-DROW، فایلە ناوخۆییەکان و شوێنی پاشەکەوت ئامادەن. کامێرا ناکاتەوە.
- **پشکنینی کامێرا و دەست** دوای ئەوەی خۆت کامێرا دەکەیتەوە، پیشانت دەدات ئایا کێشان بە دەست ئامادەیە یان نا.

پێش دەستپێکردنی کێشان پێویست نییە ئەم پشکنینانە بکەیت؛ تەنها کاتێک یارمەتیت دەوێت بەکاریان بهێنە.

## دامەزراندنی AIR-DROW

| ئامێر | چۆنیەتی دامەزراندن |
| --- | --- |
| Android / Chromium | لە menu ـی وێبگەڕ **Install app** یان **Add to Home screen** هەڵبژێرە |
| iPhone / iPad | لە Safari، **Share → Add to Home Screen** بەکاربهێنە |
| کۆمپیوتەر | کاتێک وێبگەڕەکەت نیشانی Install پیشان دا، کرتەی لێ بکە |

## گۆڕینی زمان

لە **ڕێکخستنەکان → دەربارەی ئەپ**، **کوردی** یان **English** هەڵبژێرە. ئەپەکە هەڵبژاردەکەت لەسەر هەمان ئامێر پاشەکەوت دەکات.

## تێبینییە بەسوودەکان

- بۆ کێشان بە دەست، شوێنێکی ڕوون هەڵبژێرە.
- دەستت تەواو لە چوارچێوەی کامێرا بگرە.
- یەکەم جار پڕۆفایلی **Balanced** بەکاربهێنە؛ ئەگەر جوڵەکە زۆر بوو **Stable** هەڵبژێرە.
- پێش گۆڕینی مۆبایل یان پاککردنەوەی داتای وێبگەڕ، backup دابەزێنە.
- بۆ کارکردنی نەرمتر، پێش بەکارهێنانی کامێرا tab و ئەپی زۆر داخە.

## ئەم پەکیجەی تەواوە چی تێدایە؟

سەرچاوەی AIR-DROW، مۆدێلی پشکنراوی دەست، فایلەکانی runtime ـی ناوخۆیی، فۆنت، ئایکۆن، ڕێنماییەکان و ڕێکخستنی production هەموویان لەم پەکیجەدان. `node_modules` تێدا نییە، چونکە لە `package-lock.json` ـەوە بە دروستی دوبارە دروست دەکرێت.

## تێبینیی وەشانی v7.2.1

- پەکێجی تەواوی ئایکۆنەکانت لە ناو ئەپەکەدا دانراوە.
- ئایکۆنە پێشوەکان لە workspace، settings، actions، status و brand بە تەواوی گۆڕدران.
- زیپی تەواو ئایکۆنەکان، مۆدێلی دەست، MediaPipe runtime، فۆنت و سەرچاوەی ئەپەکەی تێدایە.
## یارمەتی

لە **ڕێکخستنەکان → دەربارەی ئەپ** پشکنینی ئەپ بکە. بۆ کێشان بە دەست، یەکەم کامێرا بکەرەوە، پاشان پشکنینی کامێرا و دەست بکە. بۆ ڕێنمایی وردتر، [ڕێنمایی بەکارهێنان](./docs/USER_GUIDE_KU.md) بخوێنەوە.
