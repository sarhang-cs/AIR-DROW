# AIR-DROW v5.2.1 — چاکسازیی دۆخی شەفاف

لە preview ـی کامێرادا `FPS`، `Online` و `CAM` تەنها دەق و خاڵی نیشانەی دۆخن. هیچ باکگراوەند، box، border، pill یان blur ـیان نییە. تەنها text-shadow ـێکی بچووک ماوە تا لەسەر وێنەی ڕوون یان تاریک بخوێندرێنەوە.

## Deploy
```bash
npm install
npm run build
```

دوای push بۆ GitHub، Vercel خۆکارانە deploy دەکات.
