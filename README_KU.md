# AIR-DROW v5.0.6 — چاکسازیی تەواوی کامێرا و UI

ئەم وەشانە بۆ ئەو کێشانەی مۆبایل دروستکراوە کە لە کامێرا، ئینجینی دەست، drawer، RTL/LTR، toggle و icon ـەکان دەرکەوتبوون.

## گرنگترین چاکسازییەکان

- ئینجینی دەست بە شێوەی **CPU-first** کار دەکات بۆ جێگیری زیاتر لە Android.
- مۆدێلی دەست تەنها لە فایلی لۆکاڵی پڕۆژە دەخوێندرێتەوە؛ هیچ fallback ـی دەرەکی نییە.
- ئەگەر buffer ـی مۆدێل لە WebView ڕەت بکرێتەوە، تەنها هەمان فایلی لۆکاڵی دوبارە بەکاردهێنرێت.
- هەڵەی کامێرا و هەڵەی ئینجینی دەست جیاکرانەوە؛ کاتێک preview کاردەکات، پەیامی هەڵەی کامێرا نامێنێت.
- toggle ـەکان track و knob ـی ڕاستەقینەیان هەیە لە دۆخی Off و On.
- لۆگۆی سەرەوەی Settings و دوگمەی داخستن جێگیریان هەیە لە کوردی و ئینگلیزی.
- arrow و dropdown ـەکان شوێنی تایبەتیان هەیە و لەگەڵ نوسینی RTL تێک نادەڵێن.
- drawer ـەکە header + tabs + scroll ـی سەربەخۆی هەیە؛ هیچ card ـێک لە خوارەوە ون نابێت.
- Undo و Redo هەردووکیان SVG ـی لۆکاڵی جیاوازن.

## Build

```bash
npm install
npm run build
```

فایلی مۆدێلی دەست:

```text
web/vendor/models/hand_landmarker.task
```
