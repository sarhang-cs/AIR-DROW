# AirDraw Dynamic Logo — Lottie JSON

`airdraw-logo-lottie.json` فایلی Lottie ـی ڕاستەقینەیە و لە app ـەکە بە `lottie-web` بار دەکرێت.

- هیچ وێنەی دەرەکی یان asset ـێکی دەرەکی پێویست نییە.
- logo ـەکە SVG renderer ـی Lottie بەکاردەهێنێت.
- ئەگەر JavaScript یان Lottie بار نەبوو، ناونیشانی `AD` وەک fallback نیشان دەدرێت.
- بە toggle ـی Dynamic Logo دەتوانرێت وەستێنرێت/دەستپێبکرێتەوە.
