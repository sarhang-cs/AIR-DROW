# AIR-DROW v7.2.0 — UI Clarity Edition

## A calmer, clearer mobile experience

- Rebuilt the action-button language so primary buttons always use high-contrast text.
- Removed permanent glow from actions, selected tabs and cards; feedback now appears only during focus, hover or press.
- Replaced the inconsistent Android switch appearance with one AIR-DROW switch for every on/off control.
- Replaced duplicated dropdown triangles with one clear chevron.
- Reduced dense nested-card styling in Create, Settings and About.
- Kept camera-live status subtle, so it no longer looks like a second selected drawing tool.
- Simplified App readiness and Hand drawing check into friendly, non-technical language.
- Stopped the hand-drawing check from showing a low score before the person chooses to open Camera.
- Kept AI feedback inside AI Studio; it no longer covers the drawing canvas or exposes billing, API-key, deployment or server wording.
- Hid connection controls intended for setup from the normal user workspace.
